initialize (line 59)

Initializes component by loading all configuration files from

all plugins found in application. Configuration files should be placed in \app\plugins\your_plugin\config\ directory. Be careful, it will overwrite all settings loaded from \app\config if the setting name matches. At the end it will execute an 'initialize' callback method loaded from \plugins\your_plugin\{your_plugin}_auto_loader.php file
