getMethods (line 152)

getMethods

Gets all current methods for object with documentation. Returns an array with the following structure 'name' => name of the method 'access' => level of access to the method 'args' => array of arguments that the method takes. Args are removed from comment['tags'] as they don't need to be repeated 'comment' => keyed array see cleanComment

  • return: multi-dimensional array of methods and their attributes
  • see: $this->_parseComment
  • access: public

_parseComment

Cleans input comments of stars and /'s so it is more readable. Creates a multi dimensional array. That contains semi parsed comments
